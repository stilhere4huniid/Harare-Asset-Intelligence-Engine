## 1. Navigation
The sidebar is the main control center. Use the **Radio Button** to switch between profiles:
* **Terrace Africa (Operational):** For managing existing assets.
* **WestProp (Development):** For planning new developments.

## 2. Using the Terrace Africa Profile
* **Filter by Asset:** Use the multiselect dropdown to isolate specific malls (e.g., Highland Park).
* **Lease Expiry Slider:** Drag the slider to filter tenants based on months remaining on their lease.
* **Risk Toggle:** Check "Show High Risk Tenants Only" to instantly view the critical watchlist.
* **Generate Report:** Click the button to download a PDF summary of your current view.

## 3. Using the WestProp Profile
* **Simulation:** Check "Simulate: Close All Pipeline Deals" to see what happens if all negotiations succeed.
* **Budget Filter:** Enter a value (e.g., 3,000,000) to filter the list for high-value Anchor tenants.
* **Visuals:**
    * **Bar Chart:** Shows the gap between Committed vs. Vacant space.
    * **Table:** Lists the specific tenants meeting your criteria.

## 4. Resetting the Dashboard
Click the **"ðŸ”„ Reset Dashboard"** button at the top of the sidebar to clear all filters, simulations, and selections, returning the tool to its default state.
